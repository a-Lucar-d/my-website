ESP32 TCP Server Project

A lightweight TCP server built on ESP32

Project Overview

This project showcases how to build a TCP server using the ESP32 microcontroller. Clients on the same network can communicate with it using tools like Netcat. The ESP32 runs in Access Point + Station mode and dynamically connects to routers.
